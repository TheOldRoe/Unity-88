Thank you for purchasing Low Poly Seasonal Landscape!

This low poly stylized pack consists of 54 separate models with 2 texture palettes!

There are 2 materials, the base material, and the alternative material.  
The base material has the default colors, the alternative material has some fall colors, sand rock textures, different flower textures, etc.

Required Packages to import for Demo Scene:
*Unity Standard Assets -> Environment Package (Specifically for the Basic Water Asset)

Model Types:
*Trees and Snow Trees
*Basic Bushes
*Bare Trees and Bare Snow Trees
*2 Versions of Pine Trees and Snow Pines
*Ground Cover (Cat tails, flowers, grass)
*4 Different Mountain Types
*Stones/Rocks (and snow covered stones)
*Clouds
